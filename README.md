# Задание 1

## Задача 1

### Текст задачи

Дерево содержит вещественные числа. Отбросить в каждом из них дробную часть.

### Алгоритм решения

Описана рекурсивная функция `readCount` для считывания количества элементов дерева с проверкой на корректность: принимаются только целые положительные числа, при некорректном вводе функция вызывает себя повторно. Описана функция `fillTree`, формирующая бинарное дерево поиска из `n` случайных вещественных чисел в диапазоне от 0 до 100 с точностью до двух знаков после запятой. Описана рекурсивная функция `map`, принимающая функцию и дерево: для пустого узла возвращается пустое дерево, для каждого узла применяется переданная функция к его значению и рекурсивно обходятся левое и правое поддеревья, формируя новое дерево. В `main` считывается количество элементов через `readCount`, строится дерево через `fillTree`, после чего с помощью `map` и функции `floor` к каждому узлу дерева отбрасывается дробная часть.

# Задание 2

## Задача 1

### Текст задачи

Сформировать список из узлов с одним листом. Узел является листом, если у него нет ни левого, ни правого поддерева.

### Алгоритм решения

Описана рекурсивная функция `readCount` для считывания количества элементов дерева с проверкой на корректность: принимаются только целые положительные числа, при некорректном вводе функция вызывает себя повторно. Описана функция `fillTree`, формирующая бинарное дерево поиска из `n` случайных вещественных чисел в диапазоне от 0 до 100 с точностью до двух знаков после запятой. Описана рекурсивная функция `fold`, принимающая функцию, аккумулятор и дерево: для пустого узла возвращается аккумулятор, иначе рекурсивно обходится левое поддерево, затем применяется переданная функция к текущему узлу и его поддеревьям, после чего рекурсивно обходится правое поддерево. Описана функция `oneLeafNodes`, использующая `fold` для обхода дерева: для каждого узла проверяются его поддеревья — если одно из них пустое, а другое является листом, узел добавляется в список результата. В `main` считывается количество элементов через `readCount`, строится дерево через `fillTree`, после чего вызывается `oneLeafNodes` для получения списка узлов с одним листом.
